# Docker Setup для тестов effective-mobile.ru

1. Установить Docker
   - Windows/macOS: https://www.docker.com/get-started
   - Linux: sudo apt install docker.io

2. Собрать Docker образ
   Перейти в папку проекта, где находится Dockerfile, и выполнить:
   docker build -t effective-tests .

3. Запустить тесты в контейнере
   docker run --rm effective-tests

4. Сбор отчетов Allure (опционально)
   - После запуска контейнера, если хотите посмотреть отчет Allure, можно скопировать папку allure-results на хост:
     docker cp <container_id>:/app/allure-results ./allure-results
   - И запустить:
     allure serve ./allure-results

Примечание:
- Флаг --rm удаляет контейнер после завершения тестов.
- Все зависимости уже внутри контейнера, поэтому не нужно устанавливать Python или Playwright на локальной машине.
